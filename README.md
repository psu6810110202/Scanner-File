# Scanner-File (v1.1)

โปรแกรมสแกนหาไฟล์ประสิทธิภาพสูงสำหรับ Windows ที่รวมพลังของ Asynchronous Programming และ Multiprocessing เข้าด้วยกัน

---

## ✨ คุณสมบัติเด่น (Key Features)

● 🌍 Multi-Drive Scanning: เลือกสแกนทุกไดร์ฟในเครื่อง (C:, D:, E: ...) หรือเลือกเฉพาะไดร์ฟที่ต้องการ  
● ⚡ High Performance: แบ่งงานคำนวณ Hash ให้ CPU ทุกตัวช่วยกันทำพร้อมกัน (Parallel Computing)  
● 🔍 Smart Search: ค้นหาไฟล์ด้วย "ส่วนประกอบของชื่อ" และไม่สนตัวพิมพ์เล็ก-ใหญ่  
● 🛡️ Integrity Check: คำนวณค่า SHA-256 เพื่อเช็กว่าไฟล์เสียหรือถูกดัดแปลงหรือไม่  
● 🚫 Smart Filter: ระบบข้ามไฟล์ระบบและไฟล์ขยะอัตโนมัติ (Windows, AppData, .git)  

---

## 🏗️ รายละเอียดทางเทคนิค (Technical Details)

1. การค้นหาพิกัดไฟล์ (File Traversal)
   - ใช้โมดูล: os.walk
   - หน้าที่: ตะลุยหาพิกัดไฟล์ในโครงสร้างโฟลเดอร์

2. การคำนวณลายนิ้วมือไฟล์ (Hash Calculation)
   - ใช้โมดูล: hashlib (SHA-256)
   - หน้าที่: สร้างรหัสตรวจสอบความถูกต้องของเนื้อหาไฟล์

3. การจัดการภารกิจ (Task Management)
   - ใช้โมดูล: asyncio
   - หน้าที่: จัดการ Flow การทำงานให้ลื่นไหล ไม่หยุดรอ

4. การประมวลผลขนาน (Parallel Processing)
   - ใช้โมดูล: Concurrent Futures (ProcessPool)
   - หน้าที่: แบ่งงานหนักให้ CPU ทุกตัวช่วยกันทำพร้อมกัน

5. การบันทึกรายงานเบื้องหลัง (Background Logging)
   - ใช้โมดูล: threading
   - หน้าที่: เขียนรายงานลง scan_report.txt โดยไม่กวนการทำงานหลัก

---

## 🚀 วิธีการใช้งาน (How to Use)

1. เปิดโปรแกรมขึ้นมา (Run หรือ กด F5 scanner_file.py )
2. เลือกโหมด: กด 1 (สแกนทั้งหมด) หรือ 2 (ระบุไดร์ฟเอง เช่น C, D)
3. ใส่ชื่อไฟล์: พิมพ์ชื่อที่ต้องการหา (ใส่หลายชื่อได้โดยคั่นด้วยคอมม่า ,)
4. รอผลลัพธ์: โปรแกรมจะแสดงพิกัดและค่า Hash บนหน้าจอ และบันทึกใน scan_report.txt
